Games

Balance It Out: An algebra game written in Kaboom.js. Add, remove, and reposition weights on the scale in order to find the value of the unknown weight. Make sure to keep the scale balanced!

Equation Blaster: Solve algebraic equations to destroy enemy ships. Make sure to use the correct ammo, or it will shoot back at you! Written in Kaboom.js.

Hunt the Wumpus: A graphical version of the 1970's text-based game, written in React. Navigate a hazardous maze and look for patterns in order to hunt a legendary monster.
